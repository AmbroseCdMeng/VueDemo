数组是双向绑定：修改数组，渲染结果会立即更新
        push()
        pop()
        shift()
        unshift()
        splice()
        sort()
        reverse()

    ---------------------------------------------------------
    以上方法会改变原有数组，导致渲染结果更新。
    有几个方法不会改变原有数组
        filter()
        concat()
        slice()

    **********************************************************
    Vue 在监测数组变化时，并不是全部渲染整个列表，而是最大化的复用旧的 DOM 元素

    需要注意的是：
        以下两种情况数组变动时，Vue 是监测不到的
        1、通过索引直接修改项。app.books[3] = {name: 'VueJs'}
        2、修改数组长度。app.books.length = 2

        如果需要监测。可以使用 set 方法
            Vue.set(app.books, 3, {
                name : 'VueJs'
            })
            或
            this.$set(app.books, 3, {
                name : 'VueJs'
            })

            app.book.splice(3, 1, {
                name:'...'
            })
