v-model 的使用。
    用户表单类元素上状态和值的双向绑定。

    ★ 选中之后 v-model 的值和 value 的值是相等的

    v-model 的修饰符常用的有三个：
        .lazy
            输入框中，v-model 默认是在 input 事件中同步输入框数据（中文输入时的等待时 v-model 的特殊处理）
            使用该修饰符延迟事件同步，会转变为在 change 事件中同步数据，即当失去焦点时。
        
            .number
            该修饰符会将输入的类型转为 number 类型。
            有时候虽然输入的是纯数字，但提交或者运算时会是字符串，如:"3452"
            这时候使用该修饰符就比较有用。会将 "3452" 转换为 3452

        .trim
            过滤两端空格
