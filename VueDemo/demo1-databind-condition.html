Vue 在渲染元素时，出于效率考虑，会尽可能的复用已有的元素而非重新渲染

如下例，定义了两个 template 中各有一个 input，但是输入内容后，点击切换，发现输入的内容并未改变，仅仅是 label 和 placeholder 发生了改变

这就说明 input 被 Vue 复用了

如果不需要这种复用，可以给 input 绑定一个 key 值。 key 值必须唯一，当绑定了不同的 key 值，Vue 就不会复用

还有一个 v-show 效果和 v-if 差不多。但是 v-show 是将元素的 display 属性修改为隐藏，而 v-if 是根据条件适当的销毁和重建元素。

频繁的改变条件时，推荐 v-show 。因为 v-if 如果频繁的重建销毁，开销比较大
