Render 函数最大的缺点在于当子节点嵌套过多时，阅读起来相当困难。

为了更好的书写和阅读，Vue 提供了插件 babel-plugin-transform-vue-jsx 来支持 JSX 语法。

JSX 是一种看起来像 HTML 但实际上是 JS 的语法扩展

HTML & createElement & JSX 的对比

HTML

        <my-component :level="1">
            <span>HTML的写法</span>
            文本内容
        </my-component>

Render createElement

        new Vue({
            el: '#ex1',
            render(createElement){
                return createElement('my-component', {
                    props:{
                        level: "1",
                    }
                },[
                    createElement('span', 'createElement 的写法'),
                    '文本内容'
                ])
            }
        })

JSX -- 需要 webpack 中配置插件并编译后才可运行

        new Vue({
            el: '#ex1',
            render(h){
                return(
                    <my-component :level="1">
                        <span>JSX的写法</span>
                        文本内容
                    </my-component>
                )
            }
        })
