v-model 的使用。
    用户表单类元素上状态的双向绑定。

文本输入框

        v-model 的双向绑定效果 与 input 事件最大的不同点在于：
        input 事件是实时触发更新的
        v-model 其实是一个特殊的语法糖，它会在不同的表单上智能处理。
        比如，在中文输入法时，没有选定汉字之前，v-model 是不会更新数据的，敲下汉字时才会更新
        而 input 是实时更新的

单选按钮

        单选按钮在单独使用时不需要 v-model，直接使用 v-bind 或者 : 绑定一个 bool 值，为 true 选中，false 不选中。
        如果需要组合按钮来实现互斥效果，就可以配合 v-model 使用

复选框

        复选框不论是单独使用还是组合使用，都需要用 v-model 绑定。

选择状态: {{checked}}

下拉列表

        单选和复选框中。匹配的都是 value 值。
        在下拉列表中，option 是备选项，如果含有 value 属性，会优先匹配 value 属性，如果没有，会匹配 option 中的 text
